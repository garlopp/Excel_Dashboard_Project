# 💼 Data Science Salary Calculator (Excel)

**Author:** Victor Ntekpere

An interactive **Excel salary calculator and data analysis project** that estimates median salaries for data-related roles based on **job title, country, and job type**.

This project demonstrates practical **data cleaning, validation, and Excel-based analysis** using a real-world style dataset of global job postings, and showing my competence using Excel  

Because messy data exists everywhere… even in job listings.

---

# 📊 Project Overview

This project uses a dataset of **data science and analytics job postings** to build an interactive **salary estimation tool in Excel**.

Users can select different job parameters, and the workbook dynamically calculates the **median salary based on filtered data**.

The project demonstrates core **data analysis skills in Excel**, including:

- Data cleaning
- Data validation
- Data filtering
- Salary aggregation
- Interactive calculation tools

---

# 🧠 How the Salary Calculator Works

The **Salary Calculator** sheet allows the user to estimate salaries by selecting key parameters.

### Step 1
Open the **Salary Calculator** sheet.

### Step 2
Select a **Job Title** from the dropdown list.

Example:
```
Data Analyst
Data Scientist
Data Engineer
Senior Data Scientist
Machine Learning Engineer
```

### Step 3
Select a **Country**

Example:
```
United States
Canada
Australia
France
Mexico
```

### Step 4
Select a **Job Type**

Example:
```
Full-time
Part-time
Contractor
Internship
Temp work
```

### Step 5
The calculator automatically returns the **estimated median salary** based on the dataset.

# 📂 Dataset Description

The dataset includes job posting data with the following fields:

- `job_title_short`
- `job_title`
- `job_location`
- `job_country`
- `job_schedule_type`
- `salary_year_avg`
- `salary_hour_avg`
- `company_name`
- `job_skills`

The dataset contains thousands of job postings used to calculate **median salary insights across roles and regions**.

---

# 🧹 Data Preparation

Before building the calculator, the dataset required cleaning and structuring:

- Removing inconsistent salary entries
- Handling missing salary values
- Standardizing job titles
- Organizing countries and job types
- Creating validation lists for dropdown selections

These steps ensure the calculator produces **reliable salary estimates**.
